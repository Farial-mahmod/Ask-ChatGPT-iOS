# Ask-ChatGPT-iOS

### Introduction:
"Ask ChatGPT iOS" is an iOS app to ask ChatGPT and get AI-powered answers.

### Technologies Used:
1. SwiftUI
2. Alamofire
3. Combine
4. OpenAI API

### Tested On:
1. iPhone 14 Plus on MacOS Ventura
